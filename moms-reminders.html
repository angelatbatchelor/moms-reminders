👨‍👩‍👧 Family Sync Setup

Connect a free Firebase account so you and your mom see the same reminders on any phone. Takes about 5 minutes.

1
Open Firebase Console
Go to console.firebase.google.com → sign in with Google → click Add project → name it anything → click through to Create project
2
Create a Realtime Database
In your new project click Build → Realtime Database → Create database → pick any location → choose Start in test mode → Enable
3
Get your keys
Click the ⚙️ gear → Project settings → scroll to Your apps → click </> Web → nickname it anything → Register app
Copy the apiKey and databaseURL values and paste them below.
